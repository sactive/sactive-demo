# sactive-demo
Demo for building web with [`sactive-web`](https://github.com/sactive/sactive-web) and [`sactive-di`](https://github.com/sactive/sactive-di).

## Usage
```bash
# install dependencies
yarn install # or npm install

# start
./run.sh
```

## API for testing

- /sbot-svc/asynctest
- /sbot-svc/version
- /sbot-svc/deploy
- /sbot-svc/undeploy
- /sbot-svc/start
- /sbot-svc/stop
